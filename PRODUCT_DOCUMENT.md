# CareerPass 产品说明文档

## 一、作品简介

### 项目背景

1. **企业招聘效率低**：随着人工智能技术的发展，招聘行业正面临新的变革机遇。然而当前招聘流程仍然高度依赖简历、学历与面试表现，企业需要通过有限的信息推测求职者的真实能力，仅通过短暂的面试无法深度了解求职者，导致招聘效率低、筛选成本高以及人才错配等问题。

2. **求职者难以被看见**：对于求职者而言，尤其是应届毕业生和职场新人，往往存在"能力强但经历不足"的困境。大量优秀人才因缺少名校背景、大厂经历或简历包装能力而失去展示机会。

CareerPass希望通过AI重构招聘流程，从"经历驱动招聘"转向"能力驱动招聘"，打造AI时代的职业能力认证体系。

### 核心理念

项目提出"职业化身（Career Avatar）+ AI职业挑战（Career Challenge）+ 能力通行证（Career Pass）"三位一体的新模式，让企业招聘能力而非简历，让求职者通过真实表现获得机会。

## 二、市场调研

### 当前招聘市场存在的问题

#### 企业侧问题

1. **简历筛选效率低**：大型企业校招期间单个岗位可收到数千份简历，HR难以进行深入评估，只能通过学历、学校和企业背景进行快速筛选。

2. **面试成本高**：传统招聘需要经过简历筛选、笔试、一面、二面、终面等多个环节，时间成本与人力成本巨大。

3. **能力判断失真**：简历描述存在夸大包装现象，企业难以判断候选人的真实能力水平。

#### 求职者侧问题

1. **学历壁垒明显**：能力优秀但学校背景普通的学生难以获得面试机会。

2. **缺少能力证明渠道**：求职者只能通过项目经历或比赛经历间接证明能力。

3. **求职过程低效**：大量投递简历但难以获得反馈，形成高投入低回报的求职体验。

### AI时代的新机会

随着大模型能力的发展，AI已经具备模拟工作环境、分析用户行为、评估解决方案和生成能力画像的能力。

因此，招聘流程可以从"看经历"转变为"看表现"。

**传统招聘**：简历 → 筛选 → 笔试 → 面试 → Offer

**CareerPass**：职业化身 → AI挑战 → 能力通行证 → 业务面试 → Offer

## 三、方案介绍

### 产品架构

CareerPass由三大核心模块组成：

1. **Career Avatar（职业化身）**
2. **Career Challenge（职业挑战）**
3. **Career Pass（能力通行证）**

### 模块一：Career Avatar

用户导入简历、作品集、项目经历等职业数据。

AI自动构建职业化身。

生成内容包括：
- 能力雷达图
- 技能成长路径
- 职业方向预测
- 能力缺口分析

帮助用户建立完整职业画像。

### 模块二：Career Challenge

企业不再发布传统JD，而是发布真实工作挑战。

例如：
- 产品经理挑战：设计一款校园招聘产品
- 运营挑战：提升社区用户活跃度

用户进入AI模拟工作环境完成任务。

AI将扮演：
- Leader
- 用户
- 同事
- 面试官

模拟真实工作场景。

### 模块三：Career Pass

系统根据挑战过程进行多维度评估，形成标准化能力认证体系。

认证内容包括：产品设计能力 / 用户研究能力 / 创新能力 / 逻辑分析能力 / 沟通协作能力，最终生成Career Pass能力通行证。

企业可直接根据能力等级进行人才筛选。

## 四、核心功能与创新点

### 创新点一：用AI职业挑战替代简历筛选

**传统招聘**：简历 → 筛选 → 面试

**CareerPass**：挑战 → 能力认证 → 面试

企业不再通过经历推测能力，而是通过表现验证能力。

### 创新点二：能力通行证体系，建立统一能力标准

例如：产品设计 A+ / 用户研究 A / 数据分析 B+

企业可直接设置能力门槛，提高招聘效率。

### 创新点三：AI职业化身，构建动态成长档案

记录用户长期成长过程，形成持续更新的的人才画像。

### 创新点四：招聘流程前置验证

在进入面试前完成能力验证，帮助企业缩短招聘周期，减少无效面试。

评估维度包括：创新性 / 逻辑性 / 执行力 / 沟通能力 / 专业能力

输出标准化能力标签。

## 六、应用场景

1. **大学生求职**：通过挑战获得能力认证，绕过学历限制，获得面试机会。

2. **跨行业转岗**：通过完成目标岗位挑战证明能力，降低转岗门槛。

3. **企业校招**：通过统一挑战筛选候选人，降低HR工作量。

4. **企业社招**：快速识别高匹配人才，提高招聘质量。

## 十、总结

CareerPass并非简单优化传统招聘流程，而是重新定义AI时代的人才评价方式。

通过职业化身记录成长，通过职业挑战验证能力，通过能力通行证建立信任。

**最终实现**：让企业招聘能力，而非招聘简历；让人才凭借真实表现，而非背景标签获得机会。

这将推动招聘行业从"经历驱动"迈向"能力驱动"的新时代。
